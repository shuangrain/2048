# 遊戲說明

每次控制所有方塊向同一個方向運動, 兩個相同數字的方塊撞在一起之後合並成為他們的和, 每次操作之後會在空白的方格處隨機生成一個2或者4, 最終得到一個'2048'的方塊就算勝利！

# 模式

1. Classic：如果格子填滿且無法相加或移動時結算。
2. Time：如果數字成功相加秒數會加一上限十秒，剩下秒數為零時結算。
3. Move：如果數字成功相加步數會加一上限五步，剩下步數為零時結算。
4. Block：隨機產生一格障礙物，且會隨著數字移動，如果格子填滿且無法相加或移動時結算。
